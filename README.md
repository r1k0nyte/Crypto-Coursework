# Cryptographic Algorithms Research

Coursework project dedicated to the analysis of symmetric cryptographic algorithms.

## Algorithms

The project analyzes:

- AES
- DES
- ChaCha20

## Implemented Research

The following tests were implemented:

- Encryption time analysis
- Decryption time analysis
- File size impact analysis
- Ciphertext entropy calculation
- Avalanche effect analysis
- Memory usage analysis
- Keyspace and cryptographic strength analysis
- Graph visualization

## Technologies

The project was implemented using:

- Python 3
- PyCryptodome
- Matplotlib
